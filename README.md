# ShoppingTerminal

## Состав

- ShoppingTerminal.WPF — клиентское приложение
- ShoppingTerminal.API — Web API
- ShoppingTerminal.Data — библиотека работы с БД

## База данных

Используется SQL Server LocalDB.

Для создания БД:

Запустить проект, после чего EF Core создаст БД автоматически.
